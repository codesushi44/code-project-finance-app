# code-project-finance-app

Project Philosophy 

Based on my personal experience, many financial apps on the market are overly professional and can be quite intimidating for fresh graduates or beginners who are just starting to track their expenses. As a student of Software Engineering and Cybersecurity, I aim to combine my academic knowledge with real-life experience to develop a user-friendly desktop accounting app.

This is my first independent project on GitHub outside of my Final Year Project (FYP). Since I have a full-time job, I will be building this project step-by-step during my spare time.

Core Features:
1)Asset Snapshot: Quick overview of your current financial status.
2)Goal Tracking: Monitor your progress toward specific financial milestones.

Tech Stack

Language: Python

UI Framework: Flet (Flutter for Python)
